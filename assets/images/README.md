# Image Assets

Place your images in this directory:

## Required Images

1. **profile.jpg** - Your profile photo (recommended: 400x400px, square)
2. **project-pms.jpg** - Preventive Maintenance System screenshot
3. **project-goorca.jpg** - GoorCa API project image
4. **project-attendance.jpg** - Attendance system screenshot
5. **project-raja.jpg** - Raja Ferry integration image
6. **project-mpay.jpg** - MPay payment integration screenshot
7. **project-portfolio.jpg** - Portfolio website screenshot

## Image Guidelines

- **Format**: JPG or PNG
- **Size**: Optimize for web (under 500KB per image)
- **Dimensions**: 
  - Profile: 400x400px (square)
  - Projects: 800x600px (4:3 ratio) or similar
- **Tools for optimization**:
  - [TinyPNG](https://tinypng.com/)
  - [Squoosh](https://squoosh.app/)
  - [ImageOptim](https://imageoptim.com/)

## Fallback

The website includes SVG fallbacks if images are missing, so it will still work without images. However, adding real images will greatly improve the visual appeal.

## Adding Images

Simply drag and drop your images into this folder, making sure the filenames match those listed above.
